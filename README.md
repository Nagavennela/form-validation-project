# Simple Login Form with Validation

This is a basic login form built using HTML and JavaScript.

## 🚀 Features
- Input fields for Name, Number, and Email
- Validation for empty fields
- Error messages shown below inputs
- Simple alert on successful submission

## 🛠️ Technologies Used
- HTML
- JavaScript

## 📂 Project Structure
- index.html
- script.js (optional if separate)

## 🎯 Purpose
This project is created for learning basic form validation in JavaScript.

## 💡 Future Improvements
- Email format validation
- Phone number validation
- Better UI design with CSS
